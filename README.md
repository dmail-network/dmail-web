# dmailv2

Dmail mailbox V2

## before
### install dfx and run dfx
- install: `sh -ci "$(curl -fsSL https://sdk.dfinity.org/install.sh)"`
- see version: `dfx --version`
- enter root: `cd dmail`
- run: `dfx start`
- run: `dfx deploy`

## Available Scripts

In the project directory, you can run:

### develop
`yarn start`

### production
`yarn build`
